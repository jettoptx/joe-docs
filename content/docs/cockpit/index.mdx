The **OPTX Cockpit** is an interactive operator dashboard that provides real-time visibility into the entire OPTX infrastructure — from XRPL entry to Solana settlement. It features 7 tabbed diagram views, a live status bar, and a command bar for dispatching tasks to AstroJOE.

**Repository**: [github.com/jettoptx/optx-cockpit](https://github.com/jettoptx/optx-cockpit)

## 7 Views

### 1. Bridge Overview
Full bridge architecture: XRPL entry → Aaron Router → Wormhole NTT → Solana AgenC → astroknots Vault. Circle USDC flows through MPO Tempo x402 payment loops.

### 2. XRPL Entry
XRPL entry leg: r9 wallet → Xaman signing → Circle USDC → TraderJoe Swarm (4 agents: passive LP, lag monitor, Hooks, grid). xSPECTAR NFT soul binding for agent identity.

### 3. Aaron + PoI
Aaron Router + Proof of Intelligence layer. AGT Proof sphere processes COG/EMO/ENV tensors via zkWASM. AstroJOE orchestrates from Jetson NemoClaw.

### 4. Wormhole NTT
19 Guardian nodes validate cross-chain messages. 13/19 quorum required for VAA signatures. Native USDC burn/mint — no wrapped tokens.

### 5. Solana + AgenC
AgenC Hub on CorsairOne coordinates SubJOE agents with xSPECTAR soul bindings. MetaplexJOE handles registry. Helius webhooks monitor on-chain events.

### 6. JTX Validators
JTX staking nodes generate yields. Rewards flow to astroknots.space vault. MPO Tempo x402 handles agent-to-agent payments via Circle USDC.

### 7. AgenC Docker
Docker orchestration hub on CorsairOne. TraderJoe containers managed via AgenC protocol. Connected to AstroJOE on Jetson, Wormhole Relayer, and NemoClaw sandbox.

## Command Bar

The command bar at the bottom dispatches tasks to AstroJOE via the hermes-optx-api:

- Type a task description → sends to `POST /api/tasks`
- Type "swarm: [goal]" → sends to `POST /api/tasks/swarm`
- Recent commands shown as clickable badges

## Node Modals

Click any node in a diagram to open a detail modal showing:
- Live status indicator (green/yellow/red)
- Key-value data pairs
- **"Send Task to astroJOE"** button — dispatches a task related to that node
- **"On-Chain"** button — links to the on-chain explorer
