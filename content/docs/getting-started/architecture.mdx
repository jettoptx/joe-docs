## Edge Hardware

All private compute runs on a **Jetson Orin Nano** edge node:

- **AARON Router** — FastAPI gaze session manager + Solana bridge
- **JOE Agent** — Autonomous AI agent with Matrix chat + tool execution
- **HEDGEHOG** — MCP gateway routing AI queries through Grok, storing gaze data in SpacetimeDB
- **SpacetimeDB** — Real-time database with reducers + subscriptions
- **Conduit** — Matrix homeserver for federated agent communication

Raw biometric data never leaves this device. Only 32-byte opaque proofs reach the blockchain.
