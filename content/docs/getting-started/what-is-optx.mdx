**OPTX** (Optical Program Technologic Extension) is a privacy-preserving spatial encryption protocol for the agentic web. It combines gaze biometrics, DePIN attestation, and cross-chain token bridging on Solana.

## Core Primitives

| Primitive | Description | Endpoint |
|-----------|-------------|----------|
| **Gaze Verification** | Integrate AGT biometric auth with 3-tensor classification (COG/EMO/ENV) | `/api/gaze/store` |
| **Agent Wallet** | ERC-8004 non-transferable soulbound wallet for computational identity | `/api/wallet` |
| **CSTB Attestation** | On-chain proof of gaze with DePIN reward distribution | `/api/attestation` |

## Naming Hierarchy

| Name | Full Form | Role |
