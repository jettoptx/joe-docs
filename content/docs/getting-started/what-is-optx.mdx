**JettOptx (OPTX)** powers **JettChat**, which now operates in two modes:

- **xChat Native** — original experience with X/Twitter OAuth and faster onboarding.
- **Phantom Mode** — maximum-privacy experience powered by **Secure-Legion** (Tor + post-quantum crypto + hardware biometrics + Jett Hub).

Both modes share the same core: **JETT Auth** with the **AGT** Gaze Triad (COG/EMO/ENV), **Jett Hub** visual SSO (primary in Phantom Mode), **JTX** token gating, and end-to-end encrypted messaging with gaze-cursor typing.

**Phantom Mode adds:** triple Tor `.onion` hidden services, X25519 + ML-KEM-1024 post-quantum cryptography, StrongBox/TEE-backed keys, Ping-Pong biometric wake protocol, and a duress PIN that triggers cryptographic wipe. Implementation: [Secure-Legion/android](https://github.com/Secure-Legion/android).

## Core Primitives

| Primitive | Description | Surface |
|-----------|-------------|---------|
| **Gaze Verification** | AGT biometric auth with 3-tensor classification ([COG/EMO/ENV](/docs/authentication/gaze)) | Both modes |
| **JETT Auth** | Unified auth supporting [xChat Native + Phantom Mode](/docs/authentication/jett-auth) | Both modes |
| **JTX Gating** | [4-tier access](/docs/token/tiers) on Solana mainnet (hold 1 JTX, or stake / subscribe) | Both modes |
| **CSTB Attestation** | On-chain proof of gaze with DePIN reward distribution | Both modes |
