## Token Standards

- **$OPTX** and **$CSTB** use the **Token-2022 (SPL)** standard with transfer hooks for DePIN reward distribution
- **$JTX** is a standard SPL token on mainnet for governance
- **LayerZero OFT** wrapping enables cross-chain bridging to EVM chains (Base, Ethereum)
