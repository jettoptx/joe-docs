## Token Standards

- **$OPTX** and **$CSTB** use the **Token-2022 (SPL)** standard with transfer hooks for DePIN reward distribution
- **$JTX** is a standard SPL token on mainnet, used for [JettChat tier gating](/docs/token/tiers)
- OPTX is **Solana-native**. See [On-Chain Bridge](/docs/on-chain-bridge) for attestation-only bridging policy.
