## Architecture Layers

| Layer | Description |
|-------|-------------|
| **Edge Layer** | Biometric data stays private. Only hashes leave the device. |
| **Protocol Layer** | AARON validates proofs and routes x402 payments. |
| **Chain Layer** | Solana stores attestation PDAs. Public verification. |
