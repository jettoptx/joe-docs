A complete navigable index of the OPTX documentation graph. Every page is classified by its **Agentive Gaze Tensor (AGT)** distribution — <span style={{color: "#eab308"}}>COG</span> (Cognitive), <span style={{color: "#f43f5e"}}>EMO</span> (Emotional), or <span style={{color: "#60a5fa"}}>ENV</span> (Environmental). The **Connected** column shows all linked nodes in the [Map of Augments](/docs/dojo/moa).

**Classification framework:**
- <span style={{color: "#f43f5e"}}>**EMO**</span> = Connections, network, identity, users, bridges
- <span style={{color: "#60a5fa"}}>**ENV**</span> = Physical infrastructure, hardware, edge nodes, DePIN, topology
- <span style={{color: "#eab308"}}>**COG**</span> = Conceptual, protocol specs, algorithms, API references

See the interactive version at the **[Map of Augments](/docs/dojo/moa)**.
