> **Status: Roadmap.** The ERC-8004 soulbound agent wallet is a planned identity layer; no contract is deployed today. Treat code samples below as design intent. For current JettChat auth, see [JETT Auth](/docs/authentication/jett-auth).

The OPTX Agent Wallet (planned) implements a **non-transferable, soulbound identity** bound to gaze biometrics. Unlike standard wallets, the **ERC-8004** interface links computational proof to the holder's identity.

## Wallet Integration

```typescript
// Connect Phantom or OKX wallet
import { useWallet } from "@solana/wallet-adapter-react"

function OPTXWallet() {
  const { publicKey, signTransaction } = useWallet()

  // Mint $OPTX on devnet
  const mintOPTX = async () => {
    const mint = new PublicKey("4r9WoPsRjJzrYEuj6VdwowVrFZaXpu16Qt6xogcmdUXC")
    const program = new PublicKey("91SqqYLMi5zNsfMab6rnvipwJhDpN4FEMSLgu8F3bbGq")

    // Call DePIN program to mint tokens
    // Requires valid CSTB attestation (min 2/5 sessions)
    const tx = await program.methods
      .mintOptxReward({ amount: new BN(1000) })

        authority: publicKey,

      })
      .transaction()

    await signTransaction(tx)
  }
}
```
